Place the following two files in this directory before running build.bat:

1. plink.exe
   Source: https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html
   License: MIT (free to redistribute)
   Direct link: https://the.earth.li/~sgtatham/putty/latest/w64/plink.exe

2. vncviewer.exe
   Source: https://github.com/TigerVNC/tigervnc/releases (latest release)
   License: GPL v2
   Download the file named: vncviewer64-X.Y.Z.exe
   Rename it to: vncviewer.exe
   The LICENSE_TIGERVNC.txt in this folder covers its redistribution.

Both files are bundled into EngelRV.exe by the build script.
Customers do NOT need to install anything separately.
